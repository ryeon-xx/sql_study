# 프로그래머스 SQL 문제풀이

프로그래머스 SQL 고득점 Kit 및 코딩테스트 문제를 풀이하고 기록하는 레포지토리입니다.
SQLD 자격증 및 백엔드 개발자 취업 준비를 위해 매일 1~2문제씩 풀이하며 학습 과정을 정리합니다.

## 🎯 목적

- SQL 문법 및 쿼리 작성 능력 향상
- JOIN, 서브쿼리, GROUP BY/HAVING 등 핵심 개념 체화
- SQLD 자격증 대비

## 📁 폴더 구조
```
sql_study/
    ├── src/main/java/
    ├── select/         # SELECT 기본 문제
    ├── join/           # JOIN 문제
    ├── groupby/        # 집계 함수, GROUP BY
    ├── subquery/       # 서브쿼리
    └── string_date/    # 문자열, 날짜 처리
    └── README.md
```

## 📝 문제풀이 목록

| 번호 | 문제명 | 난이도 | 분류 | 풀이날짜 | 비고 |
|------|--------|--------|------|----------|------|
| 1 | 동물의 분류 | Lv.2 | JOIN | 2026-06-20 | LEFT JOIN, IS NULL |
| 2 | 자동차 평균 대여 기간 구하기 | Lv.4 | GROUP BY | 2026-06-22 | HAVING + 서브쿼리 |

## 💡 학습 기록

각 문제 풀이 후 막혔던 부분이나 새로 배운 개념을 간단히 기록합니다.

> 예: GROUP BY 사용 시 SELECT 절에는 집계함수나 GROUP BY에 명시된 컬럼만 올 수 있다는 걸 몰라서 에러 났음.
